Python Strings

Defining Strings


my_string1 = 'hello'
my_string2 = "hello"
my_string3 = '''Hello'''
my_string4 = """Hello"""
my_str = 'programming'

print(my_string1)
print(my_string2)
print(my_string3)
print(my_string4)
print('my_str=', my_str)
Output:
hello
hello
Hello
Hello
my_str= programming

Slicing Strings


print('my_str[0]=', my_str[0])
print('my_str[-1]=', my_str[-1])
print('my_str[1:5]=', my_str[1:5])
print('my_str[5:-2]=', my_str[5:-2])
print(my_str[:2])
print(my_str[::2])
print(my_str[::-1])
Output:
my_str[0]= p
my_str[-1]= g
my_str[1:5]= rogr
my_str[5:-2]= ammi
pr
pormig
gnimmargorp

Handling Strings with Digits


string_filter_digits = "Hello 321 World 123"
result_digits_check = all(char.isdigit() for char in string_filter_digits)
print(result_digits_check)
result_digits_check = any(char.isdigit() for char in string_filter_digits)
print(result_digits_check)
Output:
False
True

Built-in Functions for Strings


myVarForMaxMin = "HelloWorld89"
print(f"Max: {max(myVarForMaxMin)}")
print(f"Min: {min(myVarForMaxMin)}")
print(f"Type: {type(myVarForMaxMin)}")
print(f"Length: {len(myVarForMaxMin)}")
print(f"ID: {id(myVarForMaxMin)}")
Output:
Max: r
Min: 8
Type: <class 'str'>
Length: 12
ID: (some number, will vary)

as_integer_ratio() for float


myFloatingNumber = float("5.5")
print(f"Ratio: {myFloatingNumber.as_integer_ratio()}")
print(f"Ratio type: {type(myFloatingNumber.as_integer_ratio())}")
Output:
Ratio: (11, 2)
Ratio type: <class 'tuple'>
